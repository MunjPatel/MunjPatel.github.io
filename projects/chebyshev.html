Chebyshev Pricing & Risk Engine

Quantitative Research · Python

Overview

A Streamlit-based engine that computes probabilistic price bounds using Chebyshev's Inequality and log-normal moment analysis for robust, distribution-free financial forecasting. It provides a non-parametric approach to detect tail-risk and "Black Swan" events.

Key Features

  • Non-parametric pricing model using Chebyshev's Inequality
  • Tail-risk and Black Swan event detection for ad-hoc risk analysis
  • Probability bounds visualization for multi-asset portfolios
  • Evaluation of methodologies for risk-adjusted returns
  • Model reliability assurance in fast-paced market environments

Technologies

Python, Streamlit, NumPy, Pandas, Plotly, yfinance
