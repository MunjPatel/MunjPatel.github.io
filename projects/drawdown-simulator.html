Drawdown Risk Simulator

Monte Carlo · Risk Modeling

Overview

A lightweight, explainable financial risk assessment tool that simulates future stock price paths using Monte Carlo methods, detects volatility regimes via unsupervised clustering, and estimates short-term drawdown probabilities using rule-based statistical heuristics. Built with real-world market data from Yahoo Finance.

Key Features

  • Monte Carlo simulation projecting possible stock price paths
  • Volatility regime detection (stable vs. volatile) using unsupervised clustering
  • Drawdown risk estimation over short horizons
  • Interactive Streamlit + Plotly visualization
  • Fully explainable, parameterized model—no black-box ML

Technologies

Python, NumPy, Pandas, yfinance, scikit-learn, Plotly, Streamlit, Geometric Brownian Motion
