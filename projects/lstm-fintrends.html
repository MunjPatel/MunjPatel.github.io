LSTM-FinTrends

Deep Learning · Time Series

Overview

An end-to-end project that forecasts the directional movement of stock indices based on derived metrics using an LSTM (Long Short-Term Memory) deep learning model. Demonstrates a complete workflow from data preprocessing to model prediction, storage, and visualization.

Key Features

  • Data collection for global indices using yfinance
  • LSTM model for temporal dependencies in time series
  • Derived metrics: close direction, percentage changes, rolling averages, RSI
  • Flask web interface for results visualization
  • GitHub Actions automation for scheduled runs

Technologies

Python, LSTM, yfinance, Flask, GitHub Actions
